Open data e "smart disclosure": il caso della dichiarazione dei redditi

L'applicazione permette di automatizzare la compilazione della dichiarazione dei redditi (la demo presenta una versione semplificata del modello 730).

Viene utilizzato il framework OAuth2.0.

L'Authorization Server è stato implementato come un modulo PHP in Zend Framework da Antonio Pastorino. L'applicazione client permette indirizzare l'utente allo Authorization Server affinchè questo dia l'autorizzazione per i dati da scaricare e impiegare nella compilazione automatica della dichiarazione dei redditi.

I dati che servono per la compilazione sono memorizzati presso vari Resource Server:

  1. Resource Server del Comune;
  2. Resource Server dell'Agenzia delle Entrate;
  3. Resource Server dell'Agenzia del Territorio;
